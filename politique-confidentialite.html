Données collectées

Nous collectons les données personnelles suivantes via le formulaire de contact :

  • Nom complet
  • Adresse email
  • Numéro de téléphone (optionnel)
  • Type de projet
  • Message/description du projet

Finalité du traitement

Les données collectées sont utilisées pour :

  • Répondre à vos demandes de renseignements
  • Établir un devis pour votre projet
  • Communiquer avec vous concernant votre demande
  • Améliorer nos services

Base légale du traitement

Le traitement de vos données personnelles repose sur votre consentement, que vous nous donnez en soumettant le formulaire de contact. Vous pouvez retirer ce consentement à tout moment.

Destinataires des données

Vos données personnelles sont uniquement destinées à Paulygone. Nous ne vendons, ne louons, ni ne partageons vos données avec des tiers, sauf obligation légale.

Le formulaire de contact utilise le service Web3Forms pour l'envoi des messages. Web3Forms agit en tant que sous-traitant et s'engage à respecter le RGPD.

Durée de conservation

Vos données sont conservées pendant la durée nécessaire au traitement de votre demande, puis :

  • 3 ans pour les demandes de renseignements sans suite
  • Durée du projet + 5 ans pour les clients (obligations comptables)

Passé ce délai, vos données sont supprimées ou anonymisées.

Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données personnelles
  • Droit de rectification : corriger vos données inexactes ou incomplètes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la limitation : limiter le traitement de vos données
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la portabilité : récupérer vos données dans un format structuré

Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données personnelles contre la destruction accidentelle ou illicite, la perte accidentelle, l'altération, la diffusion ou l'accès non autorisés.

Cookies

Ce site n'utilise pas de cookies de traçage publicitaire. Seuls des cookies techniques strictement nécessaires au fonctionnement du site peuvent être déposés.

Hébergement

Ce site est hébergé par GitHub Pages (GitHub, Inc., États-Unis). Les serveurs sont sécurisés et GitHub s'engage à respecter les normes de sécurité en vigueur.

Transfert de données hors UE

L'hébergement sur GitHub Pages peut impliquer un transfert de données vers les États-Unis. GitHub est certifié EU-US Privacy Shield et applique des clauses contractuelles types pour garantir un niveau de protection adéquat.
